### vite, cdn, cli로 설치 

### 🔍 테스트 실행 가이드

- cli 실행
- 해당페이지에서 명령어를 입력한다.

```bash
npm run serve
```

- vite 실행
- 해당페이지에서 명령어를 입력한다.

```bash
npm run dev
```

- cdn 실행
- html에 스크립트 삽입


```html
<!-- vue3 -->
<script src="https://unpkg.com/vue@3/dist/vue.global.js"></script>
```
```html
<!-- vue2-->
<script src="https://cdn.jsdelivr.net/npm/vue/dist/vue.js"></script>
```

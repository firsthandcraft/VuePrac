## :bulb:서비스 및 프로젝트 소개

**`vuedongsan`** 는 vue로 제작한 오늘의 집 디자인을 참고로 구성한 상품리스트 원페이지 입니다. 
- 해당상품을 클릭하면 모달차잉 나오면서 해당상품의 제목이 나옵니다.
- 좋아요를 누를 수 있는 기능을 추가했습니다.
- 상품에 대한 데이터를 data.js에서 가져왔습니다.

## :wrench: 사용 기술 스택
-vue 3.0
